This plugin adds the buttons to Burmese unicode fonts Ayar to zawgyi vice versa, Ayar Online editor, Fonts convertor, Dictionary and other ..

== Description ==
Automatic unicode font converter for Ayar and zawgyi zawgyi vice versa, Ayar Online editor, Fonts convertor, Dictionary and other userful tools for burmese language.

The Ayar Unicode Toolbar feature list:<ul>
<li>Allow to change Ayar font and zawgyi font.</li>
<li>Ayar Online Editor</li>
<li>Fonts convertor</li>
<li>Dictionary</li>
</ul>

== Installation ==
* Just put the plug-in into your plug-in directory and activate it.
